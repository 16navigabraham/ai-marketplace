✅ STEP 1: TURSO DATABASE (SQLite Serverless)
────────────────────────────────────────────────────────────────────────────
[ ] Go to: https://turso.tech
[ ] Sign up (GitHub)
[ ] Install CLI: choco install turso
[ ] Run: turso auth signup
[ ] Create DB: turso db create ai-marketplace
[ ] Get URL: turso db show ai-marketplace --url
    → Copy to backend/.env as DATABASE_URL
[ ] Get Token: turso db tokens create ai-marketplace
    → Copy to backend/.env as DATABASE_AUTH_TOKEN

Free Tier: ✅ Perfect for hackathon
Cost: $0/month (very generous limits)


✅ STEP 2: UPSTASH REDIS (Cache)
────────────────────────────────────────────────────────────────────────────
[ ] Go to: https://upstash.com
[ ] Sign up (GitHub/Google)
[ ] Create Redis database
[ ] Copy connection string
    → Paste into backend/.env as REDIS_URL
[ ] Format: redis://your-namespace-xxxxx.upstash.io:xxxxx

Free Tier: ✅ 10,000 commands/day (enough for testing)
Cost: $0/month


✅ STEP 3: ALCHEMY RPC ENDPOINTS (Blockchain Read Access)
────────────────────────────────────────────────────────────────────────────
[ ] Go to: https://www.alchemy.com
[ ] Sign up (GitHub/Google)
[ ] Create app for ETHEREUM
    → Get API key
    → Copy to backend/.env as ETH_RPC_URL
    → Format: https://eth-mainnet.g.alchemy.com/v2/YOUR_KEY
[ ] Create app for POLYGON
    → Copy to backend/.env as POLYGON_RPC_URL
[ ] Create app for ARBITRUM
    → Copy to backend/.env as ARBITRUM_RPC_URL
[ ] Create app for BASE
    → Copy to backend/.env as BASE_RPC_URL

Free Tier: ✅ 300 million compute units/month (plenty)
Cost: $0/month


✅ STEP 4: VENICE AI (Optional - LLM)
────────────────────────────────────────────────────────────────────────────
[ ] Go to: https://venice.ai
[ ] Sign up
[ ] Get API key from dashboard
[ ] Copy to backend/.env as VENICE_API_KEY

Free Tier: ✅ Available

✅ STEP 5: UPDATE backend/.env
────────────────────────────────────────────────────────────────────────────

Replace ALL xxx values with:
